What is Staci?
  • Staci is a freely available software package for hydraulic simulation of pipeline systems developed jointly by the Dept. of Hydrodynamic Systems and Sys-Team Co. Ltd.
  • The software is suitable for steady-state analysis of hydraulic pipe networks:
    • mass flow rates and velocities in the pipes,
    • nodal pressures,
    • water depth distribution in the open-surface channels,
    • residence time distribution (fluid age) and
    • chlorine distribution.
  • The library of Staci includes the most common hydraulic system elements such as pipes, reservoirs, valves, pumps and channels.
  • You can track the change of the water depth in the reservoirs during a simulation period.
  • In the case of open-surface channel flow, the software automatically switches between pressurized and open-surface flow.

Hogyan működik a program?
  • A Staci két részből áll:
    • A számításokat végző program (számdaráló). Ez a C++ kód a hidraulikai modell implementációját és megoldóját tartalmazza. A Staci szerveren fut, így nem kell a felhasználó gépére telepíteni.
    • A grafikus felhasználói felület (megjelenítő). Ez a felhasználó gépén fut és a szerveren elvégzett számítások vizuális megjelenítését, valamint a szerverrel való kommunikációt végzi.
How does it work?
  • Staci consists of two main parts:
    • The number-crunching software. This C++ code includes the implementation of the hydraulic model and its solver. As it runs on the Staci server, there is no need to install it on the user's PC.
    • The graphical user interface (GUI) written in Java. This runs on the user's PC and takes care of the communication with the server and the visualization of the results.

  • A program használatához Java futtatási környezet (JRE) szükséges.
  • Felhasználónevet ezen a címen lehet igényelni.

  • Részletes leírás a dokumentációban található.
How do I use the software?
  • First of all, you'll need Java Runtime Environment (JRE).
  • You'll also need an account, send an email to this address.
  • The software itself can be found here: Launch Staci!
  • Although the GUI (graphical user interface) can be used in itself, to run computations, you'll need a working internet connection, as the computations themselves are performed by the Staci server.
  • Detailed description can be found in the documentation.
